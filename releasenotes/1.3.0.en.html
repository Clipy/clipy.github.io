Clipy 1.3.0

The first official update in approximately eight years.

This update includes major internal changes, including a database migration, so that development can continue going forward. Existing data will be migrated automatically as much as possible, but migration may fail depending on your environment.
Please export important data, such as snippets, before updating.

This release focuses on making Clipy usable and stable on current macOS environments, including Apple Silicon support and internal improvements. We plan to continue improving Clipy and fixing bugs in future updates.

Enhancements

  • Added support for macOS 13 Ventura and later.
  • Added support for Apple Silicon.
  • Improved preference persistence and window behavior.
  • Adjusted the pasteboard monitoring interval to improve copy stability.
  • Improved handling of sensitive information from password managers and similar apps.

Bug Fixes

  • Fixed a problem where whitespace was removed when importing snippets.
  • Fixed repeated login item notifications.
  • Fixed an unnecessary accessibility permission prompt.
  • Fixed thumbnail aspect ratio handling.
  • Fixed layout problems in dark mode.
  • Fixed inline menu numbering.

To Our Users

Thank you for using Clipy for such a long time. This update has taken approximately eight years, but we plan to continue development going forward. We also plan to work on dark mode support and other feature improvements, so please look forward to future updates.

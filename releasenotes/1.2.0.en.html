Clipy 1.2.0

  • Support macOS 10.14 Mojave.
    • Accessibility permission is required to execute paste behavior from macOS 10.14.
    • Please allow Clipy access to accessibility from "System Preferences" → "Security & Privacy" → "Privacy" → "Accessibility".
  • Paste operation now works properly even when using keyboard other than QWERTY layout.
  • Added German localization.
  • Added Simplified Chinese localization.
  • Fixed bug.
